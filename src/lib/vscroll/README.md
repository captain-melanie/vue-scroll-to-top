# v-scroll-to-top

## Demo
[demo](https://captain-melanie.github.io/vue-scroll-to-top/dist/index.html)

## Installation

### npm
```$xslt
npm i v-scroll-to-top
```

## Usage
```$xslt
import Vue from 'vue
import VScrollToTop from 'v-scroll-to-top'

Vue.use(VScrollToTop)
```

##### SampleComponent.vue
```$xslt
<template>
  <v-scroll-to-top></v-scroll-to-top>
</template>
```
